API Endpoints for the peer reviewers to test:
/restaurant/ - Main restaurant page
/restaurant/booking/tables/ - List and manage booking appointments
/restaurant/menu/ - List and manage menu items
/restaurant/menu/<menu_item_id>/ - Display and manage a single menu item
/auth/users/ - List, manage and register new users
/auth/token/login/ - Generate login token for an user.
/restaurant/api-token-auth/ - Alternative version of the above endpoint (Generates login tokens)
/auth/token/logout/ - Logout the current user
